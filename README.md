# Bookbot

A simple command-line tool that analyzes text files to report word counts and character frequencies.

## Usage
Run the script from your terminal by passing the path to a text file:
```bash
python3 main.py books/frankenstein.txt
```
##Features
Word Count: specific count of total words found.
Character Stats: frequency of each letter, sorted from most to least common.
Case Insensitive: converts all text to lowercase for accurate counting.
